we are currently in mitch user , mitch user have permission to run vim without password , there is a feature in vim , we can execute a command for example 

$vim -c pwd

-c tells vim to run the following pwd command , if you run this command it will print current working directory , so using this method we can call /bin/bash using sudo . on result we spawn a shell.
